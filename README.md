# EVA

## Запуск проекта

Установить зависимости

```
NODE_ENV=development npm ci`
```

Создать файл `.env.local` в корне проекта со следующим содержимым

```
BACKEND_ORIGIN=https://divan17.intranet.hhw.ru
```

Запустить проект в режиме разработчика

```
npm run dev
```

Открыть в браузере

```
http://localhost:4444
```
